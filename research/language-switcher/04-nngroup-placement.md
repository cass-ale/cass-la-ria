# NNGroup — 6 Tips for Improving Language Switchers
Source: https://www.nngroup.com/articles/language-switching-ecommerce/
Date: March 2022

## Key Findings from User Research

1. **Upper right corner is the expected location** — Users across cultures (US, Chinese) look there first
2. **Top left corner is the secondary location** — If not found in top-right
3. **Globe icon is universally recognized** — Users seek icons that represent language
4. **Don't use flags alone** — Flags represent countries, not languages (Swiss speak 4 languages)
5. **Label languages in their native script** — "Français" not "French"
6. **Persist the choice** — Don't reset on page reload

## For Cass la Ria
- Place globe icon in the **top-right or bottom-right corner** of the decorative border
- Since the site is a single centered layout, bottom-left or bottom-right of the frame works well
- Small, unobtrusive — matches the minimalist aesthetic
- On hover: show the language list
- On click: select and persist
